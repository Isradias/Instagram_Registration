Instagram Registration Page Clone

Este projeto é uma réplica da página de registro do Instagram, criada exclusivamente para fins de estudo e demonstração de HTML e CSS.

📌 Objetivo

O objetivo deste projeto é apresentar a estrutura e o estilo de uma página de registro semelhante à do Instagram, sem funcionalidades de backend ou armazenamento de dados.

🛠️ Tecnologias Utilizadas

HTML5

CSS3
